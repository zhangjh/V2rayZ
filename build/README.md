# Build Resources

此目录用于存放应用图标等构建资源。

## 所需文件

- `icon.ico` - Windows 应用图标（256x256 或更高）
- `icon.icns` - macOS 应用图标
- `icon.png` - Linux 应用图标（512x512）

## 图标制作

可以使用以下工具制作图标：

- Windows: 使用 IcoFX 或在线工具将 PNG 转换为 ICO
- macOS: 使用 `iconutil` 命令行工具或 Image2Icon 应用
- Linux: 直接使用 PNG 格式

## 注意事项

图标应该清晰、简洁，在不同尺寸下都能保持良好的视觉效果。
