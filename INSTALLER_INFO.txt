欢迎安装 V2rayZ
Welcome to V2rayZ Setup

这是一个现代化的 Windows v2ray 客户端，支持 VLESS 协议、智能分流和系统托盘集成。
This is a modern Windows client for v2ray with VLESS protocol support, smart routing, and system tray integration.

系统要求 / System Requirements:
- Windows 10 (64-bit) 或更高版本 / or higher
- 至少 200 MB 可用磁盘空间 / At least 200 MB free disk space
- 至少 4 GB RAM / At least 4 GB RAM

主要功能 / Key Features:
✓ VLESS 协议支持 / VLESS protocol support
✓ 全局代理模式 / Global proxy mode
✓ 智能分流（国内直连）/ Smart routing (China direct)
✓ 自定义域名规则 / Custom domain rules
✓ 系统托盘集成 / System tray integration
✓ 流量统计 / Traffic statistics
✓ 现代化 UI / Modern user interface

注意事项 / Important Notes:
• 本软件需要管理员权限来设置系统代理
  This software requires administrator privileges to set system proxy
  
• 首次运行时，请配置您的 VLESS 服务器信息
  On first run, please configure your VLESS server information
  
• 如果遇到问题，请访问 GitHub 仓库提交 Issue
  If you encounter issues, please visit our GitHub repository to submit an issue

项目地址 / Project URL:
https://github.com/v2rayclient/v2ray-windows-client

感谢使用！/ Thank you for using V2rayZ!
